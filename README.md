# dynamodb-utils

## Function List

- 一括削除／一括登録
- 全件検索／条件付き検索
- 単件登録／更新／削除
- ログ出力制御
- CLI の提供
  - 一括削除
  - 一括登録

## Dynamodb Supported Function

| Name       | Description    |
| ---------- | -------------- |
| Scan       | 全件検索       |
| Query      | 特定条件で検索 |
| Get        | １件取得       |
| Put        | １件更新       |
| Delete     | １件削除       |
| BatchWrite | 複数件更新     |
| BatchGet   | 複数件取得     |
| Truncate   | 全件削除       |
| Bulk       | 全件登録       |
